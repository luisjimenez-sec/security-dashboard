# Security Dashboard

Herramienta de monitoreo de seguridad de red desarrollada en Python y Bash.

## Que hace?
- Escanea la red local y detecta todos los dispositivos conectados
- Identifica fabricantes y sistemas operativos de cada dispositivo
- Guarda logs automaticos de cada escaneo con fecha y hora
- Ejecuta escaneos automaticos cada hora mediante cron

## Tecnologias utilizadas
- Python 3
- Bash
- Nmap
- Linux (Kali)
- Git / GitHub

## Uso
sudo python3 scanner.py
sudo bash scripts/escaneo_automatico.sh
